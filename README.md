Check-response
===============

PHP-Javascript system to check http server response.

## Set up:

* Interval between requests
* Fetch content or just headers
* User-Agent
* Keep-Alive
* Cookie
* Accept-Encoding
* Accept-Charset
* Accept-Language

Single and multiple (splitted) modes.
